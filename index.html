验证二叉搜索树

发表于 2024-02-29 分类于 代码随想录 ， LeetCode
本文字数： 728 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/validate-binary-search-tree

二叉搜索树（又称二叉排序树）的特点如下：

  • 左子树 < 根结点 < 右子树
  • 中序遍历可得到有序序列
  • 从根结点开始比较，若关键字小于根结点则进入左子树，否则进入右子树
  • 可使用递归，但执行效率低（空间复杂度高）；非递归使用循环
  • 结点的插入：插入的结点一定是叶子结点，可使用递归实现；若存在相同的关键字，插入失败

这题使用中序遍历得到序列，然后判断该序列是否是升序的

错误思路：递归判断每个子树是否满足“左孩子 < 根结点 < 右孩子”，每个子树是满足该条件的，但是下面这种情况就会误判

从中序与后序遍历序列构造二叉树

发表于 2024-02-29 分类于 代码随想录 ， LeetCode
本文字数： 880 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/construct-binary-tree-from-inorder-and-postorder-traversal

唯一地确定一棵二叉树：

  1. 先序和中序：先序确定根结点，中序确定左右子树
  2. 后序和中序：同上
  3. 层序和中序：层序遍历确定根结点

显然这题还是使用递归的思想，中序遍历是“左-中-右”，后序遍历是“左-右-中”，根据左右子树的顺序特点确定二叉树，思路如下：

  • 先写好退出递归的条件，即当序列数组为空时，返回NULL
  • 然后开辟一个节点的空间，将当前的根节点（即后序遍历序列最后一个元素）存放在开辟的空间中
  • 遍历中序遍历序列找到该根节点的位置，从该处分为左右子树，并将当前根节点的左右指针指向两个子树
  • 接着分别对左子树和右子树进行上述操作（注意拆分后的序列的边界）

路径总和

发表于 2024-02-28 分类于 代码随想录 ， LeetCode
本文字数： 764 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/path-sum/

这题用到了回溯的思想，若使用迭代法，则每次回溯时需要减去刚才保存的节点值，再继续；而递归本身就有回溯的功能

  • 当遍历到叶子节点时，比较路径上的节点值之和与目标值是否相等
  • 若不相等则返回false，回到上一层继续遍历
  • 若相等则返回true，这里定义两个bool类型变量存放左、右路径的布尔值

注意这里不能直接定义bool l = preOrder(root->left, targetSum, path);，同时需要注意对l和r进行初始化。

找树左下角的值

发表于 2024-02-28 分类于 代码随想录 ， LeetCode
本文字数： 885 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/find-bottom-left-tree-value

这题用迭代法更方便，迭代法思路：每次记录当前层的第一个节点，直到最后一层。

我使用的是递归法，思路如下：

  • 先定义一个全局变量（结构体）用于存放最深的层数和该层最左边节点的值

  • 使用前序遍历，按照“根-左-右”顺序遍历二叉树

  • 按照该顺序遍历，若当前节点为叶节点，则比较深度大小判断是否需要记录当前节点值

    这里不需要判断是左叶子节点还是右叶子节点，因为同一层的叶子节点会先记录左节点，然后再遍历右节点，此时深度值已经更新，所以同一层的右节点不会记录下来

  • 若当前节点有孩子节点，则继续遍历

二叉树的所有路径

发表于 2024-02-27 分类于 代码随想录 ， LeetCode
本文字数： 1.1k 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/binary-tree-paths

使用深度优先搜索——先序遍历，通过递归不断回溯，找到所有的路径

  • 先判断root是否是空节点
  • 然后判断root的孩子节点
  • 若孩子节点不为空，则将当前节点值加入sta数组（记录路径,可看作栈），然后分别遍历当前节点的左子树和右子树
  • 若孩子节点为空，说明当前节点是叶节点（并没有加入sta数组），使用sprintf拼接路径，然后再将当前叶节点加入路径
  • 然后将当前路径加入路径数组ans

spritnf返回了本次函数调用最终打印到字符缓冲区中的字符数目

平衡二叉树

发表于 2024-02-27 分类于 代码随想录 ， LeetCode
本文字数： 638 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/balanced-binary-tree/

利用递归的思想：

  • 从最深的两个子树开始判断
  • 每次判断两个子树的高度差是否大于1
  • 若小于等于1，则返回上一层，同时记录当前子树高度（左右子树高度中的较大值）
  • 若大于1，则返回-1（若左右子树高度为-1，则一直返回-1）
  • 重复上述步骤

二进制求和

发表于 2024-02-26 更新于 2024-02-27 分类于 代码随想录 ， LeetCode
本文字数： 1.3k 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/add-binary/

这题可以通过模拟二进制求和的步骤来实现：

  • 先为结果数组分配空间，注意分配的空间要比最大数组还多两个字节，因为两个数相加可能有进位，同时max_length不包括'0'，所以要预留一个字节。

  • 定义flag用于判断是否有进位

  • 使用for循环从后往前遍历两个数组，将对应字符转为整数，然后将两个数字和flag进行相加，得到sum

  • 若sum%2 == 1，则当前位置为'1'；若sum/2 > 0，则有进位

  • 循环结束后，将数组最后一位置为'\0'，表示数组结束

  • reverse()函数用于将上述结果数组进行倒序，得到正确答案

    big[l1 - i - 1] - '0'表示big[l1-i-1]字符对应的整数
