回文数

hxt
2023-07-07
LeetCode › 数据结构
C, LeetCode, 数据结构

题目地址：https://leetcode.cn/problems/palindrome-number/

给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

  • 例如，121 是回文，而 123 不是。

提示：

  • \(−2^{31}≤X≤2^{31}−1\)

通过这题我发现了我对for循环的运行顺序的理解不正确（说到底还是基础不牢😓）

  • for循环的运行顺序

    i=0 ——> i<100 ——> printf ——> i++ ——> i<100 ——> printf ——> i++ ...

    最后打印的i的值为99，而i的真实值为100。
