根据 x 开方后是否是整数分为两种情况：

  • 能找到一个整数 k 使得 k2 <= x，使用二分法一定能找到该整数

  • x 平方根 k 为浮点数，且k ∈ (n, n+1)，其中n为整数，所以k的整数部分ans即为n

    因此需要在if判断语句中不断更新ans的值，使其取得更大的值；若是在while循环结束时才给ans赋值，则可能会导致ans等于n+1
