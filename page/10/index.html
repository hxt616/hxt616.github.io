有效的括号

发表于 2024-02-09 分类于 代码随想录 ， LeetCode
本文字数： 1.9k 阅读时长 ≈ 2 分钟

题目地址：https://leetcode.cn/problems/valid-parentheses/

有关括号匹配的题一般都是用栈来实现。

下面是我自己的代码，比较繁琐：

  • 遍历 s 字符串，遇到左括号就入栈，遇到右括号则判断栈内是否为空，若不为空则判断栈顶元素是否为对应的右括号；若不满足条件则返回 false ，满足条件则栈顶元素出栈
  • 循环结束后判断栈内是否为空，若不为空（有多余的左括号）则返回 false

下面介绍代码随想录的解法：

遍历 s 字符串，遇到左括号，则入栈；遇到右括号则与栈顶元素进行配对

  • 第一种情况：已经遍历完了字符串，但是栈不为空，说明有相应的左括号没有右括号来匹配，所以return false

  • 第二种情况：遍历字符串匹配的过程中，发现栈里没有要匹配的字符。所以return false

  • 第三种情况：遍历字符串匹配的过程中，栈已经为空了，没有匹配的字符了，说明右括号没有找到对应的左括号return false

用栈实现队列

发表于 2024-02-08 分类于 代码随想录 ， LeetCode
本文字数： 1.5k 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/implement-queue-using-stacks/

用两个栈实现队列，很容易想到：将第一个栈(输入栈)中的元素保存到第二个栈中，此时第二个栈(输出栈)就是原来元素形成的队列。

实现的过程中，myQueuePop()函数总是有小错误：将输入栈将元素均存入了输出栈后，输入栈此时为空，调用取出队首元素后，不对输入栈进行处理；然后调用myQueuePush()函数，将新元素存入输入栈中，此时再调用myQueuePop()函数，将输入栈中新元素再存入输出栈中，接着取出队首元素，发生错误（元素顺序不正确）。

修改后的myQueuePop()函数：先将输入栈中元素存入输出栈中，取出队首元素，然后将剩余元素重新存入输入栈中，后面在调用myQueuePush()函数不会影响到元素顺序。

三数之和

发表于 2024-02-08 分类于 代码随想录 ， LeetCode
本文字数： 1.8k 阅读时长 ≈ 2 分钟

题目地址：https://leetcode.cn/problems/3sum/description/

这题使用双指针法比较合适：

  • 先使用qsort()函数将数组进行升序排序

  • 由于是三数之和，所以使用一个for循环确定第一个元素下标i，初始化左右指针——使用left指针指向第二个元素（i+1），right指针指向数组最后一个元素

  • 当三数之和大于0时，执行right--，因为当前数组是有序的，所以左移右指针使得三数之和减小

  • 当三数之和小于0时，执行left++，原理同上

  • 难点在于去重，不能有重复的三元组，但三元组内的元素是可以重复的！

    1. 对一个元素去重：比较nums[i]和nums[i-1]

      若 i>0 且 nums[i]==nums[i-1]，则说明三元组中第一个数不变，又因为此时右指针不做任何操作，即三元组中有两个值不变，则第三个值也是确定的，因此产生了重复的三元组

    2. 对后两个元素去重：原理同上

重复的子字符串

发表于 2024-02-07 分类于 代码随想录 ， LeetCode ， KMP算法
本文字数： 1k 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/repeated-substring-pattern/

注：这题的意思是判断给定的字符串能否只通过它的子字符串构成而不包含其他字符串

解题思路：

  • 一个字符串能由它的子字符串构成，则该子字符串的长度不可能大于整个字符串长度的一半

  • 字符串s的最长相等前、后缀不包含的字符串就是最小重复的子串

    步骤一：因为 前缀和后缀是相等的字符串，t[0] 与 k[0]相同， t[1] 与 k[1]相同，所以 s[0] 一定和 s[2]相同，s[1] 一定和 s[3]相同，即：，s[0]s[1]与s[2]s[3]相同 。

    步骤二： 因为在同一个字符串位置，所以 t[2] 与 k[0]相同，t[3] 与 k[1]相同。

    步骤三： 因为 这是相等的前缀和后缀，t[2] 与 k[2]相同 ，t[3]与k[3] 相同，所以，s[2]一定和s[4]相同，s[3]一定和s[5]相同，即：s[2]s[3] 与 s[4]s[5]相同。

    步骤四：循环往复。

  • 如果len % (len - next[len - 1]) == 0 ，则说明数组的长度正好可以被 数组长度-最长相等前后缀的长度 整除 ，说明该字符串有重复的子字符串

    若next[len-1]==-1, 则没有最长相等前后缀，也就不可能由重复的子字符串构成

    若len-next[len-1] > len/2，则也不存在满足要求的子字符串

IPv6

发表于 2024-02-04 分类于 计算机网络
本文字数： 383 阅读时长 ≈ 1 分钟

IPv6特点：

  • 只能在源点分片，传输过程中的路由器不能分片
  • 首部长度不可变

  • 单播：一对一通信，可作为源地址和目的地址

  • 多播：一对多通信，可作为目的地址

  • 任播：一对多中的一个通信，可作为目的地址

IPv4 向 IPv6 过渡：

  • 双协议栈：既支持IPv4，又支持IPv6
  • 隧道技术：使IPv6数据报在IPv4中传输

IPv6可以与DHCP结合使用分配地址，也可以在不使用DHCP服务器的情况下也可以静态分配IPv6：

  • 使用 EUI-64（64位扩展唯一标识）将 MAC地址（48位）转换成64位，再与64位IPv6前缀形成新地址（128位）

  • MAC地址转换成 EUI-64 格式的步骤：

    1. 将二进制形式的MAC地址的第7位（又称U/L bit）进行取反，相当于将BIA[burned-in address]转换成本地管理的地址。

    2. 在MAC地址的前三个字节和后三个字节之间添加fffe

    3. 最后在前面加上IPv6的64位前缀
