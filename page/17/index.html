地址的寻址能力

16位地址的寻址能力为64KB 一个内存单位的大小是1个字节（B）。 16根地址总线可以寻址的范围为2的16次方个内存单位（单位：B）。 2^16 B = 2^10 * 2^6 B 2^10 = 1024 B = 1KB 所以 2^16 B = 1KB * 64 = 64KB 所以16位CPU的一个段的长度最大为64KB。
