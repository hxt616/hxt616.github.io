长度最小的子数组

发表于 2024-01-02 分类于 代码随想录 ， LeetCode
本文字数： 744 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/minimum-size-subarray-sum/

滑动窗口算是双指针的一种，可以类比计算机网络里的滑动窗口。

主要的思想是：从A[i]开始，向后遍历找到满足条件的最小长度；若能找到，分析可知以A[i]开始的窗口大小只有这一个最小值，然后寻找以A[i+1]开始的最小长度。

  • 初始时，左、右指针指向数组下标为0的元素，窗口大小为最大正整数
  • 右指针右移，比较当前窗口中元素之和与目标值的大小：
    1. 若大于等于target，且当前窗口小于之前的值，则更新窗口大小，同时左指针右移，直到窗口里元素之和小于target
    2. 否则，右指针继续后移
  • 最后比较窗口大小是否等于最大正整数，若等于，则说明不存在符合条件的子数组，返回0；否则返回窗口大小

这里虽然使用了while，但时间复杂度还是O(n)，不是O(n2)

有序数组的平方

发表于 2023-12-31 分类于 代码随想录 ， LeetCode
本文字数： 650 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/squares-of-a-sorted-array/

这题可直接暴力求解，即将每个元素平方后排序（快速排序），时间复杂度为O(nlogn)；

更好的方法是使用双指针的思想，由于数组为非递减有序数组，所以平方后的最大值一定在数组两端：

  • 初始时，左指针指向数组初始位置，右指针指向末尾元素
  • 比较两个指针指向元素平方后的大小，将较大的值存入新数组中
  • 若左指针指向元素较大，左指针右移；若右指针指向元素较大，右指针左移

由题目可知 x 平方根的整数部分 ans 是满足 ans2 ≤ x 。下面分析用二分法求解 k 值。

根据 x 开方后是否是整数分为两种情况：

  • 能找到一个整数 k 使得 k2 <= x，使用二分法一定能找到该整数

  • x 平方根 k 为浮点数，且k ∈ (n, n+1)，其中n为整数，所以k的整数部分ans即为n

    因此需要在if判断语句中不断更新ans的值，使其取得更大的值；若是在while循环结束时才给ans赋值，则可能会导致ans等于n+1

    注意：mid的数据类型应为long，防溢出（x 的范围是 0 ~ 231-1）
