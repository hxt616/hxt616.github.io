二叉树中的最大路径和

发表于 2024-03-18 分类于 LeetCode
本文字数： 633 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/binary-tree-maximum-path-sum/

这题看着不难，实现起来还是比较复杂，递归关系比较难想到。

  • 先确定终止条件，也就是当前节点为空，直接返回0
  • 然后记录左节点、右节点的最大路径和
  • 接着记录当前路径和（当前节点值+左节点路径和+右节点路径和）
  • 若该路径和大于之前的路径和maxS，则更新maxS
  • 最后需要返回，上一级节点以当前节点为子树，则子树最大路径和为root->val + fmax(leftM, rightM)

发表于 2024-03-12 分类于 代码随想录 ， LeetCode
本文字数： 632 阅读时长 ≈ 1 分钟

题目地址：https://leetcode.cn/problems/house-robber/

根据题目不能连续访问相邻的数组元素，因此每次间隔只能为1或2（间隔大于2的情况包含在这两种情况内，只是多跳几步）。

  • dp[i]：表示小偷在前 i 个房屋偷到的最大金额（元素之和）

  • 递推公式：间隔为1或2，即跳跃距离为2或3，取上一步的最大值加上当前房屋的金额

  • 初始化：dp[0] = 0；dp[1] = nums[0]（前1个房屋最大金额就是第一个元素）；若房屋数大于1，则dp[2] = nums[1]（也可从第二个房屋开始）

  • 遍历顺序：跳跃距离为2或3，因此for循环中的 i 应该大于等于3
